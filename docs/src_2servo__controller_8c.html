◆ servo_initialize_controller()

esp_err_t servo_initialize_controller ( const servo_controller_config_t * servo_cfg,
servo_controller_handle_t * ret_handle )

Initializes the servo controller.

Parameters
[in]servo_cfgServo controller configuration
[out]ret_handleThe returned handle for the servo controller
Returns
  • ESP_ERR_NOT_SUPPORTED rmt or rmt sync manager is not supported on this device (try S3)
  • ESP_ERR_INVALID_ARG An input parameter in servo_cfg is invalid
  • ESP_ERR_INVALID_STATE The handle has already been initialized
  • ESP_ERR_NO_MEM Not enough memory to initialize device
  • ESP_OK The controller was initialized successfully

◆ servo_set_all()

esp_err_t servo_set_all ( servo_controller_handle_t handle,
float servo_position )

Set the angle of all 32 servos.

Note
If the servo controller is currently started, the servos will begin to during the next 20ms period.
Parameters
[in]handleHandle for the servo controller
[in]servo_positionServo position. Will be set to nearest discrete angle based on servo precision. Must be between 0 and 180 degrees.
Returns
  • ESP_ERR_INVALID_ARG The input is invalid or out of range.
  • ESP_OK The servo angle has been set

Set the angle of a single servo.

Parameters

Returns

  • ESP_OK The servo angle has been set

◆ test_board()

esp_err_t test_board ( servo_controller_handle_t handle)

Used to test the pcb board by setting servo control pins. The pins should alternate between high and low or 5v and 0v.

Note
Can be tested without the board using a oscilliscope, check github for expected image.
Parameters
[in]handleHandle for the servo controller
Returns
  • ESP_ERR_INVALID_ARG The input handle is invalid.
  • ESP_ERR_INVALID_STATE The servo controller has been started.
  • ESP_OK The pins have now been set.
